# GoIT JS Homework 04 - javaScript Objects

Цей репозиторій містить рішення завдань з JavaScript для домашньої роботи 04.

## Структура проекту

- `js/task-1.js` - рішення першого завдання
- `js/task-2.js` - рішення другого завдання  
- `js/task-3.js` - рішення третього завдання
- `index.html` - HTML файл для тестування

## Запуск

Відкрийте `index.html` у браузері для тестування рішень.
